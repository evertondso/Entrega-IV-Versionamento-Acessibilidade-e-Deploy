# Entrega-III-Interatividade-e-Funcionalidades
Caminho União

O projeto Caminho União foi desenvolvido como parte das atividades práticas de aprendizado em desenvolvimento web, com o objetivo de criar um site representando uma organização sem fins lucrativos voltada à união de pessoas e ações sociais. O trabalho começou de forma simples, utilizando apenas HTML e algumas imagens para montar a estrutura básica das páginas e apresentar o conteúdo principal. Nessa fase inicial, o foco estava em compreender a marcação HTML e a construção de seções, títulos, parágrafos, listas e inserção de imagens. O site apresentava apenas o conteúdo fixo, sem estilos visuais definidos.

Com o passar das aulas e a evolução do aprendizado, o projeto passou a utilizar CSS para melhorar o visual e a experiência de navegação. Foram aplicados conceitos de cores, fontes, espaçamento e responsividade, deixando o site mais agradável e organizado. Também foram criadas classes e variáveis de estilo, o que facilitou a manutenção do código. O uso de gradientes, sombras e bordas arredondadas trouxe um aspecto mais moderno, e o design passou a se adaptar a diferentes tamanhos de tela, tornando o projeto acessível tanto em computadores quanto em dispositivos móveis. Nessa etapa, foi adicionada a identidade visual da ONG, com cores e imagens de apoio.

Após dominar as bases de HTML e CSS, o projeto avançou para a implementação de JavaScript, adicionando interatividade e funcionalidades dinâmicas. Foram criadas validações para o formulário de cadastro, garantindo que os campos fossem preenchidos corretamente e exibindo mensagens de erro ou sucesso de forma automática. Também foi implementada a alternância entre tema claro e tema escuro, utilizando variáveis CSS e controle via JavaScript, permitindo que o usuário escolhesse o modo visual de preferência. Além disso, o JavaScript passou a controlar partes do conteúdo e efeitos de transição, tornando o site mais fluido e funcional.

O projeto evoluiu não apenas tecnicamente, mas também em organização. O código passou a ser dividido em arquivos separados para facilitar a manutenção e o entendimento. A utilização do Git e do GitHub trouxe o controle de versões, permitindo registrar cada avanço, corrigir falhas e realizar o deploy do site por meio do GitHub Pages. Isso possibilitou acompanhar a evolução do trabalho de forma clara, desde as primeiras linhas de HTML até as melhorias com CSS e a inclusão do JavaScript.

No resultado final, o site Caminho União apresenta uma estrutura sólida, com design limpo, responsivo e interativo. O formulário é funcional e acessível, as seções são bem definidas e o tema escuro garante uma boa experiência visual. O projeto demonstra a progressão de aprendizado em desenvolvimento web e o domínio das principais tecnologias utilizadas na construção de um site moderno: HTML, CSS, JavaScript e Git.
